## 개요

학부때 이미 많이 학습했고, 정리하는 시간도 가졌기에 다시 정리하는건 리소스 낭비라 생각하여 키워드만 정리하겠다.

외운다기보다 순차적으로 읽으면서 흐름을 쫓아가는게 더 도움이 될 듯하다.
