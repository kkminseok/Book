## 서평

- 초보자가 읽기 좋다
- 그만큼 간편하다. 전공자가 읽기에는 조금 부족한 내용이 많다. 특히 아쉬운 부분은 동기화 부분이였다.
- 그럼에도 한 번 훑기 좋았다. 읽기 좋은 방법은 여기에 나온 키워드 중심으로 다시 찾아보는것이 좋은 것 같다.
