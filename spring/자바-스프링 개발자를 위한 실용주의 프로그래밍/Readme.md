## 책을 고른 이유

1. 회사에서 팀비가 남아서
2. 나름에 최신에 쓰인 책이고 평이 좋았다.
3. 프레임워크에 대한 학습이 필요하다기보다 좀 더 탄탄한 기반을 다지고 싶었는데 목차가 탄탄기를 다질 수 있는 목차였다.
4. 초반부만 읽었을때는 객체지향적 사고를 기르게 해줘서 좋음.
