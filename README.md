# 개요

원래 블로그에 정리했던 글들이지만, 블로그 테마의 레이아웃들이 생각보다 깔끔하게 정리되지 않고 파편화 되어 있다고 생각하여 레포를 만들어야겠다고 생각.

책 마다 레포지토리를 만드는건 너무 많아질까봐 하나의 레포지토리에 계층구조로 정리할 예정.

책을 읽어야만 기술력을 키울 수 있는건 아니므로 업데이트를 얼마나 자주할 지는 모르겠음.
